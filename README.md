# 수익모델 — 무인 SaaS 비교 어필리에이트 사이트

**목표:** 맨바닥(AI/코드만, 무자본·무오디언스)에서 "공짜점심"에 가장 가까운 수익 모델을 골라 **무인 자동 운영** 시스템을 만든다.

## 결정 로그

- **수익 모델:** ⑨ 어필리에이트 (무자본·무재고·한계비용 0·무인화 최적). 근거 영상: "돈 버는 방식은 정해져 있다 — 수익 모델 15종"(Ladder Game).
- **하위 니치(객관적 스코어링 1위):** 온라인 비즈니스 빌더 SaaS(퍼널·결제·코스·올인원). 최고 **반복 커미션**(Systeme.io 60% 평생, GoHighLevel 40% 평생 등)이 결정적.
- **아키텍처:** 프로즈 리뷰가 아니라 **자동 수집한 구조화 데이터(가격·기능) 기반 비교 사이트**.
  - 이유: 2026 구글 업데이트로 순수 AI 양산 글 제휴사이트 71% 순위 하락. 반면 "독자 데이터·실제 사용 근거"는 순위 상승. → 데이터가 해자이자 자동화 가능 영역.

## 정직한 제약 (완전 $0·완전 무인 아님)

- 도메인 ~1.5만원/년(1회 구매, 사람이). 호스팅은 무료 티어 $0.
- 제휴 프로그램 가입은 **사람이 직접**(신원·세금, 일부는 기존 트래픽 요구).
- 커미션율은 수시로 바뀜 → `tools.json`의 값은 APPROX, 가입 시 확인(`verified:false`).
- 최초 수익까지 **수주~수개월**(SEO 색인). 며칠 아님.

## 로드맵

1. ✅ **니치 확정 + 데이터 스키마 + 시드 리스트** — `data/tools.json`
2. ✅ **데이터 수집기(+Playwright 폴백)** — `scripts/collect.py` → `data/scraped.json` (12/12 수집)
2.5 🟡 **티어 정제** — `scripts/parse_tiers.py`. 결정적 추출은 **자가 인증 불가**로 판명(무자본 A방식의 한계 실측). 6/12만 신뢰 → 사람 인증본 `data/curated.json`. 나머지 6개는 백필 대상(사이트별 파서 or LLM 전환).
3. ✅ **정적 사이트 생성** — `scripts/build_site.py` → `site/` (비교표 + 툴별 페이지, 신선도 배지, 제휴 고지). 파이프라인 관통 검증됨.
4. ⬜ **제휴 링크(내 ref ID) + 지표 추적** — 핵심지표 "전환당 수익". 제휴 프로그램 가입(사람) 필요.
5. ⬜ **무인 갱신** — cron으로 2→2.5→3 자동 재실행·재배포 + 도메인/호스팅 배포.

### 백필 현황 (2.5)
- ✅ **teachable** — 사이트별 셀렉터(`SELECTORS`)로 해결, curated 추가 → **7종 라이브**
- ⚠️ samcart·thinkific — Framer 자동클래스 / 무거운 JS라 취약. 재시도 or 수동 인증 필요
- ❌ kartra·webflow·builderall — 페이지빌더 DOM/탭 구조라 **결정적 추출 비현실적**. LLM 추출로 전환하거나 수동 인증본으로만 추가 권장

## 구조

scripts/collect.py  # 2. 수집기
scripts/parse_tiers.py  # 2.5 티어 추출
scripts/build_site.py   # 3. 사이트 생성
site/               # 생성된 정적 사이트
README.md           # 결정 로그 + 로드맵
